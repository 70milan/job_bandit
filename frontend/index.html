About

Interview Assistant helps you ace technical interviews by transcribing audio, capturing screenshots, and generating AI-powered responses and coding solutions using your resume and job description context.

Why This App?

Completely self-hosted with no recurring fees. Just $1 one-time + your own OpenAI API costs (typically $0.01-0.05 per interview).

Competitors charge $50-200/month for similar features. With Interview Assistant, you own it forever and control your data.

How It Works
  1. Create a session with your resume, API key, and job description
  2. Click Start to begin the session timer
  3. Use System Audio (Ctrl+R) to capture interviewer's questions
  4. Use Microphone (Ctrl+M) to capture your own speech
  5. Use Screenshot (Ctrl+K) for instant AI solutions without saving
  6. Use Screenshot (Ctrl+S) to save context for later AI reference
  7. AI generates responses as if YOU are answering, using your resume
Tips
  • Use Ctrl+Q to minimize to stealth mode - invisible to screen sharing
  • Use Ctrl+Shift++ and Ctrl+Shift+- to resize the app
  • Use Ctrl+Alt+Arrow keys to move the app around
  • Ctrl+S saves screenshots for context, Ctrl+K gives immediate answers
  • All conversations are saved to your session folder
